# Memory Map:

| Address Range   | Size   | Description        |
|-----------------|--------|--------------------|
| `$0000-$7FFF`   | 32KB   | RAM                |
| `$8000-$83FF`   | 1KB    | VIA #1             |
| `$8400-$87FF`   | 1KB    | VIA #2             |
| `$8800-$8BFF`   | 1KB    | UART               |
| `$8C00-$8FFF`   | 1KB    | Unused             |
| `$9000-$BFFF`   | 12KB   | Expansion Port     |
| `$C000-$FFFF`   | 16KB   | ROM                |

> **Total:** 64KB
